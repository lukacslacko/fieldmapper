Field Mapper

Walk around a room with your phone — it samples the magnetic field every few centimetres, sends the readings to the server, and the server reconstructs the field as a tangle of glowing lines that stay anchored in 3D space.

WebXR mode Android Chrome

Uses ARCore via WebXR for 6 DoF tracking + the phone's magnetometer for the field. Field-line reconstruction runs on the server, so the phone only has to render and collect data.

  • Android Chrome with ARCore installed
  • Permission for camera, motion sensors, and magnetometer
  • Open this page over HTTPS (or http://localhost on the dev machine)
Start AR

Server

Heavy work — IDW interpolation + RK4 field-line tracing — runs in a Python process at the same origin as this page. Each session uploads samples in batches and polls back for the latest reconstruction.
